Welcome to our Student Affairs Website

Our platform is designed to help you manage student information in an efficient and organized manner.
With our easy-to-use functionalities, you can add, update, and delete student information, search for active students by name, and assign departments to students who meet the required criteria.

platform offers the following minimum functionalities:

  • Add New Student: You can easily add a new student to our system by providing their id, name, date of birth, GPA, gender, level, status, department, email, and mobile number.
  • Update Existing Student Information: You can update an existing student's information, except for the department field, which is shown disabled for editing.
  • Update Existing Student Information: You can update an existing student's information, except for the department field, which is shown disabled for editing.
  • Delete Student Data: Our platform allows you to delete an existing student's data with a confirmation dialogue for the action before deletion occurs.
  • Search for Active Students: You can search for active students by name in the search for students screen. Students with similar names having active status are rendered as a table.
  • Assign Department to Students: You can select a specific student after searching to assign a department through the student's department assignment page. The page includes student ID, name, and a dropdown list for available departments and a submit button. This action is applicable for students if level = 3; otherwise, an error is shown to the user with a clear, understandable error message.
  • View All Students: You can view all active/inactive students in a separate page rendered in a table with a related set of attributes only.
  • Change Student Status: You can change the status of a student from active to inactive or vice versa from the table viewing all students.
